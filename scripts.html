Gameplay: Dynamic Flashlight & Battery Management

This C# script manages the complex behavior of the player's flashlight, including procedural flickering, energy consumption, battery charging mechanics, and audio-visual feedback loops.

Battery charging & audio feedback loop.

Procedural flicker based on battery state.

1. Flicker Simulation

The flicker is handled via a Coroutine that iterates through defined FlickerSettings. It uses random intensity ranges to simulate aging hardware before triggering a full blackout period.


private IEnumerator FlickerRoutine(FlickerSettings settings) {
    _isFlickering = true;
    yield return new WaitForSeconds(Random.Range(settings.minDelay, settings.maxDelay));
    // ... Randomizing intensity loop
    SetLightIntensity(settings.flickerMinIntensityPercent);
    yield return new WaitForSeconds(Random.Range(settings.minOffDuration, settings.maxOffDuration));
    _isFlickering = false;
}
